## 🚀 Getting Started

### Quickest Start (Docker)
```bash
docker-compose up
# Frontend: http://localhost:3000
# Backend: http://localhost:5000
```

### Manual Setup
```bash
# Terminal 1 - Backend
cd backend
python -m venv venv
venv\Scripts\activate  # or source venv/bin/activate
pip install -r requirements.txt
python app.py

# Terminal 2 - Frontend
cd frontend
npm install
npm run dev
```
